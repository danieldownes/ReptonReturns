Repton Returns Editor v2

Individual Level Settings
 

 

 This panel controls the global settings for the currently selected level.

'Level name' - sets the name for the level.

'Time' - Set the time-bomb's starting count (in seconds). if '-1' is entered, no time restriction is present.

'Map from start' - sets whether Repton holds the navigational-map from the level start.

'Size X' and 'Size Y' sets the number of level tiles in the horizontal (X) direction vertical (Y) direction, once the 'Regenerate with new size' is selected.
