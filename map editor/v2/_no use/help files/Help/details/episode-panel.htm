Repton Returns Editor v2

Episode Management

 


 		 'Name' represents the name of the episode.


There are currently three different style of play, selectable from the 'Style' drop down.

'Repton 1' - All levels in the episode act like the original 'Repton 1' game; for example, the map piece type is selectable but the 'Map from start' data field is not.

'Repton 3' - All levels in the episode act like the original 'Repton 3' game; for example, the map piece type is not selectable and the 'Map from start' data field is automatically controlled so that levels 1 to 5 set the 'Map from start' to checked, and levels 6 and beyond set the 'Map from start' to unchecked.

The 'Levels' data area allows level addition, deletion and reordering control, as well as level navigation functionality.

 - An infinite* number of levels can be added using the [ + ] button.

 - Levels can be deleted by first selecting a level from the list and then using the [ - ] button. There will be at least one level per episode. Note; this command does not give confirmation. If you do anciently delete a level it can be recovered via the 'temp' folder within the editors' installed folder providing you do not first close the editor or create a level of the same name.

 - A level can be repositioned within an episode by first selecting the desired level, and then using the Up/Down buttons. The order the level appear represent the order in witch they are played.

 

* With restriction to data storage space.
