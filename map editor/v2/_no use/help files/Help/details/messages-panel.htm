Repton Returns Editor v2

In-Game Messages

 

 


 

 This give the selected level the functionality to show in-game messages triggered at predefined points in the level

[ + ] allows creation of messages

[ - ] deletes the active message

[ < / > ] allows navigation between the avible in-game messages for selected level.

'<No Messages>' is a text field to enter the message. Messages can contain new lines.

'Add/Delete' manages the triggers of when the message should activate. Once selected a location on the level should be chosen. If the chosen location is already a trigger, then that trigger is deleted. Once the message in the game has been shown it will not re-trigger, even if a different trigger location is passed.
