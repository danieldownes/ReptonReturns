Repton Returns Editor v2

Piece Type panel

 


 		 Selecting and placing pieces on a level

Select a piece type from the panel and place appropriately on the level grid. Hovering over a piece type brings up information about that piece. Rocks and eggs will fall if the piece type states 'LEFT/RIGHT slant' or 'no support'

'Clear with selected' will clear the level with the selected piece types, some piece type can not be applied to this function (E.g. start position, bombs, spirits, etc).

 

 

Scenery pieces

This data area allows addition of 'one-off' scenery contained within the map. You may add create your own scenery using a 2D and 3D graphics packages.

When selecting [ + ], you are prompted to set an area within the level of witch the scenery mesh will cover. When the player can see the selected area, the scenery will be rendered.

[ - ] will delete the active scenery piece.

[ < ] and [ > ] provide selection of the active scenery piece.

'Mesh File' allows selection of the 3D data file. This should be in the format of Microsoft's DirectX 3D file format (extension '.x"). The file should be contained within a folder named 'scenery' within the saved episode's folder, or within the 'common scenery' folder contained in the Repton Returns game 'data' folder.

'Texture File' allows selection of the 2D data file that will texture onto the selected mesh file. This should be in the format of Microsoft's bitmap file format (extension '.BMP"). The file should be contained within a folder named 'scenery' within the saved episode's folder, or within the 'common scenery' folder contained in the Repton Returns game 'data' folder.
