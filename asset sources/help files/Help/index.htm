Repton Returns v1 BETA 2

 

This is a BETA version of Repton Returns, meaning that bugs may still exist. If you come across an error, please report it;
see http://www.Ex-D.net/games/products/ReptonReturns/ for details. Thank you.

Use the standard keyboard arrow keys to move Repton around the level. Pressing the escape key while you are in a playing level will return you to the 'Home' level, pressing the escape key in the 'Home' level will return you to the main menu.

If you are interested in helping to design more levels please get in touch. Software will be provided to help build the levels, if it is decided to sell extra levels, as for example, an expansion pack, you will receive a substantial percentage.
