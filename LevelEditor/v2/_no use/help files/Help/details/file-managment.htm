Repton Returns Editor v2

Saving / Opening / Importing

Saving

When saving, a file containing the episode details will be created as well as a file for each level, therefore, when specifying a filename, you are in-fact specifying a folder name that will be created to contain these files.

From the 'File' menu choose 'Save', or if you wish to save a different version of the current episode choose 'Save As'. Navigate to where you wish to save, specify the folder name in the Filename fieild (no need to create a folder) and click save.

Opening

When opening, you must specify an episode file (extension '*.rre'), all levels associated to that episode will be loaded.

From the 'File' menu choose 'Open', then navigate and select a .rre file before clicking open.

Importing

Currently only episodes from the recent FX-Repton can be imported.

To import an FX-Repton episode (set of levels), choose 'File > Import > FX-Repton' from the menu. Select any file within the FX-Repton episode folder; files should sequentially named from 'map1.fxr' and all be in the same folder.

Levels will appear as designed, at the specified size (max 99 in x or y direction), and passwords will be decoded and form the level's name. The first level name will also become the episode name by default.
