Repton Returns Editor v2

Debugging your level
 

The editor provides information of what the currently selected level should include, or not include.

The editor currently does not check logic of a level, or check if it completable. There is a command provided ('Test level now') that will load the game with the selected level (skipping intros and menus, etc) so that you can check if it is possible to compete the level.
